越黒リタ Official Gallery — GitHub Pages 公開用

このフォルダ内の次のファイルを、GitHubリポジトリの一番上へアップロードしてください。

- index.html
- style.css
- app.js
- .nojekyll
- covers フォルダ
- icons フォルダ
- images フォルダ

公開設定

1. GitHubで新しいリポジトリを作成します。
2. 上記ファイルをアップロードして保存します。
3. リポジトリの Settings → Pages を開きます。
4. Build and deployment の Source を Deploy from a branch にします。
5. Branch を main、フォルダを /(root) にして Save を押します。
6. 数分後に表示される公開URLを開きます。
